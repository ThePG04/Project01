In Shopify Admin go to **Online Store → Themes → ... (your theme) → Edit code**. 2. Create a new file under **layout** called `theme.liquid` and paste the content from section #1. 3. Under **templates** create `index.liquid`, `collection.liquid`, `product.liquid` and paste respective contents (#2,#3,#4). 4. Under **sections** create `header.liquid`, `footer.liquid`, `hero-test.liquid` and paste (#5,#6,#7). 5. Under **snippets** create `product-card.liquid` and paste (#8). 6. Under **assets** create `theme.css` and `theme.js` and paste (#9,#10). 7. Under **config** create `settings_schema.json` and paste (#11). 8. Save everything. Go to your store preview — homepage will use `index.liquid` and show the hero + preview. --- ### QUICK CUSTOMIZATION NOTES - Colors are defined in `assets/theme.css` as CSS variables — change `--color-darkblue`, `--color-red`, `--color-purple` to tweak brand hues. - Replace `collections.frontpage` with your actual collection handle or create a collection named `frontpage`. - Replace any `Test` / `Test01` placeholder text with your real names, product handles, or collection handles. - To add categories later: create Shopify Collections and they will appear in the Collections area; use navigation editor to add links. --- If you want, I can also: - Give you a ready list of `collection handles` and `sample product handles` to paste into the templates. - Convert this into a downloadable zip file with proper folder structure.
